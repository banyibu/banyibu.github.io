C++学习笔记第四部分高级主题，17-19章总结

计算机基础,计算机语言,C++ Primer

概述

标准库特殊设施、用于大型程序的工具、特殊工具与技术。

第十七章：标准库特殊设施

tuple类型
tuple<T1,T2,...,Tn> tt是一个tuple，其包含n被初始化的值，值的类型分别是T1…Tn。
make_tuple、get<i>(t)、tuple_size<tupleType>::value、tuple_element<i, tupleType>::type
bitset类型
定义在同名头文件中。
bitset<n> b、bitset<n> b(s, pos, m, zero, one)、bitset<n> b(cp, pos, m, zero, one)
bitset操作：any() b.all() b.none() b.count() b.size() b.test() b.set(pos,v) b.set() b.reset(pos) b.reset() b.flip(pos) b.flip() b[pos] b.to_ulong() b.to_ullong() b.to_string(zero, one) os<<\b is>>b
正则表达式
正则表达式库组件：regex regex_match regex_search regex_replace sregex_iterator smatch ssub_match
regex选项：r1=re r1.assign(re,f) f.mark_count() r.flags()
一个正则表达式语法是否正确是在运行时解析的。
regex_iterator迭代器是一种迭代器适配器。，可以用其获得所有匹配。
sregex_iterator it(b, e, r); sregex_iterator end;
smatch操作：ready(); m.size(); empty(); prefix(); suffix(); format(…); length(n); position(n); str(n); m[n]; begin(); end(); cbegin(); cend()
编号为0的子表达式是整个匹配序列。
\{d}{3}三个数字序列；[-. ]一个-或.或空格。?表示可选的。
子串匹配操作：matched; first; second; length(); str(); s=ssub;
regex_replace：接受一个输入字符序列和一个regex对象和一个描述我们想要输出的字符串。m.format(dest, fmt,mft); m.format(fmt,mft); regex_replace(dest, seq, r, fmt, mft); regex_replace(seq, r, fmt, mft);
string fmt="$2.$5.$7"; // 将号码格式替换为ddd.ddd.ddd
匹配和格式标志的类型为match_flag_type。这些值都定义在名为regex_constants的命名空间中。
using std::regex_constants::format_no_copy;
匹配标志：match_default; match_not_bol; match_not_eol; match_not_bow; match_not_eow; match_any; match_not_null; match_continuous; match_prev_avail; format_default; format_default; format_sed; format_no_copy; format_first_only;
随机数
随机数引擎类和随机数分布类。
C++程序应该使用default_random_engine类和恰当的分布类对象。
随机数引擎操作：e; e(s); e.seed(s); e.main(); e.max(); Engine::result_type; e.discard(u);
分布类型和引擎：

一个给定的随机数发生器一直会生成相同的随机数序列。一个函数如果定义了局部的随机数发生器，应该将其定义为static的。否则，每次调用函数生成相同的序列。
其他随机分布：uniform_real_distribution<\double> u(0,1); normal_distribution<> n(4,1.5); bernoulli_distribution b;
IO库再探
操纵符修改流的格式。通常改变后的状态对所有后续IO都生效。
操纵符setprecision和其他接受参数的操纵符都定义在头文件iomanip中。
定义在iostream中的操纵符：boolalpha/noboolalpha showbase/showbase showpoint/noshowpoint showpos/noshowpos uppercase/nouppercase dec hex oct left right internal fixed scientific hexfloat defaultfloat unitbuf/nounitbuf skipws/noskipws flush ends endl
我们可以强制一个流使用科学计数法、定点十进制或是十六进制计数法。操纵符scientific和fixed
setw类似endl，不会改变输出流的内部状态。它只决定下一个输出的大小。
定义在iomanip中的操纵符：setfill(ch); setprecision(n); setw(w); setbase(b);
未格式化的输入/输出操作：is.get(ch); os.put(ch); is.get(); is.putback(ch); is.unget(); is.peek();
多字节底层IO操作：is.get(sink, size,delim); is.getline(sink, size, delim); is.read(sink,size); is.gcount(); os.write(source,size); is.ignore(size, delim);
seek和tell函数：tellg(); tellp(); seekg(pos); seekp(pos); seekp(off, from); seekg(off,from);
由于只有单一标记，因此只要我们再读写操作切换，就必须进行seek操作来重定位标记。
