第十一章，关联容器

1）map 或set 2）关键字重复否 3）有序否
map set multimap multiset unordered_map unordered_set unordered_multimap unordered_multiset
使用关联容器

关联容器概述
关键字类型要求，有序容器，默认需要支持<运算符。
multiset<Sales_data, decltype(compareIsbn)*> bookstore(compareIsbn);
pair上的操作：pair<T1,T2> p; pair<T1,T2> p(v1,v2); pair<T1,T2> p={v1,v2}; make_pair(v1,v2); p.first; p.second; < <= > >= == !=
关联容器操作
关联容器额外的类型别名：key_type mapped_type value_type
关联容器insert操作：insert(b,e); insert(li); insert(v); emplace(args); insert(p,v); emplace(p,args)
从关联容器删除元素：erase(k) erase(p) c.erase(b,e)
map和unordered_map的下标操作：c[k] c.at(k)
在一个关联容器中查找元素的操作：find(k) count(k) c.lower_bound(k) c.upper_bound(k) c.equal_range(k)
无序容器
无序容器管理操作：1）桶接口：c.bucket_count() c.max_bucket_count() c.bucket_size(n) c.bucket(k) ；2）桶迭代：local_iterator const_local_iterator c.begin(n) c.end(n) c.cbegin(n) c.cend(n); 3）哈希策略：c.load_factor() c.max_load_factor() c.rehash(n) c.reserve(n)
无序容器对关键字的要求：==运算符，还需要一个hash<\key_type>类型的对象来生成每个元素的hash值。

第十二章，动态内存

静态内存、栈内存、堆
动态内存与智能指针
new delete 内存泄露 指针空悬
智能指针：shared_ptr unique_ptr weak_ptr 都定义在memory头文件中。
shared_ptr 和 unique_ptr都支持的操作: shared_ptr sp; unique_ptr up; p; *p; p->mem; p.get(); swap(p,q); p.swap(q);
shared_ptr独有的操作：make_shared(args); shared_ptrp(q); p=q; p.unique(); p.use_count()
nothrow
定义和改变shared_ptr的其他方法：shared_ptr p(q); shared_ptr p(u); shared_ptr (q,d); shared_ptr p(p2,d); p.reset(); p.reset(q); p.reset(q,d);
unique_ptr操作：unique_ptr u1; unique_ptr<T,D> u2; unique_ptr<T,D> u(d); u=nullptr; u.release(); u.reset(); u.reset(q); u.reset(nullptr);
weak_ptr：weak_ptr w; weak_ptr w(sp); w=p; w.reset(); w.use_count(); w.expired(); w.lock();
动态数组
